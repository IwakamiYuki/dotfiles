# タスク

指定されたファイルを Google Chrome で開いてください。

## 引数

ユーザーの引数: `$ARGUMENTS`

## 手順

1. **ファイルパスの特定**
   - 引数が指定されている場合:
     - ファイルパスとして解釈（絶対パスまたは相対パス）
     - 「直前」「最新」「いま」「今」「さっき」「生成した」などのキーワードが含まれる場合は、会話履歴から最近 Write または Edit ツールで操作した Markdown ファイルを特定
   - 引数が指定されていない場合:
     - 会話のコンテキストから、最近生成または編集した Markdown ファイルを推測

2. **ファイルの存在確認**
   - 相対パスの場合、現在の作業ディレクトリからの相対パスとして解釈
   - ファイルが存在しない場合はエラーを報告

3. **複数ファイルの処理**
   - 特定されたファイルが複数ある場合:
     - 5 個以下の場合: すべてのファイルを自動的に開く
     - 6 個以上の場合: ファイルリストを表示してユーザーに確認してから開く

4. **Chrome で開く**
   - `open -a "Google Chrome" "<filepath>"` を実行
   - 複数ファイルの場合は各ファイルに対して実行
   - ファイルパスにスペースが含まれる場合は適切にクォート処理

5. **完了報告**
   - 開いたファイルパス（複数の場合はすべて）を報告
